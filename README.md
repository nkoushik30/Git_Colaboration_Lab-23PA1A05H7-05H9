# Git_Colaboration_Lab_05H7_05H9

## Overview
This repository demonstrates Git collaboration workflow using multiple branches and pull requests.

## Branch Structure
- **main**: Contains the main module and core functionality
- **koushik**: Contains the koushik module with user management features
- **satya**: Contains the satya module with data processing features

## Modules
- `main_module.py`: Core application entry point
- `koushik_module.py`: User management features developed by Koushik
- `satya_module.py`: Data processing features developed by Satya

## Workflow
1. Each developer works on their own branch (koushik, satya)
2. Modules are developed independently on separate branches
3. Changes are merged to main via pull requests
4. This demonstrates collaborative development using Git

## Running the Modules

### Main Module
```bash
python main_module.py
```

### Koushik Module (User Management)
```bash
python koushik_module.py
```

### Satya Module (Data Processing)
```bash
python satya_module.py
```

## Collaboration Lab Assignment
This lab demonstrates:
- Creating and managing multiple branches (main, koushik, satya)
- Developing separate modules on different branches
- Pushing different modules to their respective branches
- Merging changes through pull requests
- Collaborative Git workflow
